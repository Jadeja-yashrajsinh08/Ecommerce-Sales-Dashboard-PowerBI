# Ecommerce-Sales-Dashboard-PowerBI
Interactive Ecommerce Sales Dashboard built using Power BI.
🛒 Ecommerce Sales Dashboard (Power BI)
**📌 Project Overview**

This project is an interactive Ecommerce Sales Dashboard built using Microsoft Power BI.

The dashboard helps analyze sales performance, customer behavior, product categories, payment methods, monthly profits, and regional sales.

**Features**
Total Sales Amount,
Total Quantity Sold,
Average Order Value (AOV),
Total Profit,
Monthly Profit Analysis,
Sales by Customer,
Sales by State,
Quantity by Category,
Quantity by Payment Mode,
Profit by Sub-Category,
Quarter and State Filters,
